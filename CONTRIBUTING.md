# Contributing

Thank you for your interest in Proton Pass CLI.

## Source code availability

This source code is published for **transparency purposes only**. We believe users and security researchers should be able to inspect the code that handles their credentials and secrets.

## External contributions

We do not accept external contributions at this time. This includes pull requests, patches, issues and feature requests submitted through this repository.

If you have found a **security vulnerability**, please report it responsibly through our [bug bounty program](https://proton.me/security/bug-bounty) rather than opening a public issue.

For general feedback or support, please visit [proton.me/support](https://proton.me/support).
